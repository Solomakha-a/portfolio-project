Weather Project

This Weather App was built with HTML, CSS and React using API. It allows users to search for weather conditions in a specific city or retrieve weather information based on their current location. The app displays the city name, current temperature, weather description, humidity, wind speed, and date/time. It also provides a weather forecast for the upcoming days.
